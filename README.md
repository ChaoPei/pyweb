##### Using python3.4 create sample blog webstation
* **JinJa2**
 * CSS和HTML模板，用于产生前端页面
* asyncio
 * 异步IO
* **aiohttp**
 * 异步web底层服务 
* **mysql**
 * 数据库服务  
* **watchdog**
 * 监测文件修改变化，自动重启服务器，提升开发效率 
* **Vue**
 * MVVM框架，更好的分离Model和View 
* **markdown2**
 * 引入markdown的python实现，将博客和评论嵌入markdown中
* **Nginx**
 * web服务器、负责反向代理
* **Supervisor**
 * 管理进程，监控服务进程
* **Fabric**
 * 自动化部署工具
* **图标**
 * UIkit
